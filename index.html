1

이 파일을 GitHub에 올리기

이 index.html 파일을 GitHub 저장소(Lynna-study)에 업로드하면 돼요.

2

접속 URL

https://jaminishere-web.github.io/Lynna-study/
이 주소로 접속하면 런처가 바로 열려요!

3

새 파일 추가할 때

새 아티팩트를 GitHub에 올리고, 이 index.html에 카드 한 줄만 추가하면 돼요.

Safari에서 런처 URL 열기 → 하단 공유 버튼(□↑) → "홈 화면에 추가" 탭 → 완료!
